# Arduino-Controlled-Robot
Arduino-controlled mobile robot demonstrating sensor-based navigation and embedded control logic

Purpose:
- Moves autonomously using sensors
- Detects obstacles and changes direction
- Uses Arduino C++ for control logic

Technologies used:
- Arduino motor
- L298N motor
- C++
- Ultrasonic Sensor

Outcomes:
- How to integrate multiple hardware and software subsystems
- How to debug embedded systems
- How to structure control logic in code.
